# SQL Scripts

This directory contains all SQL scripts for the Fashion Match Game database.

## File Structure

- `01_tables.sql` - Database schema and table definitions
- `02_functions.sql` - Database functions and triggers
- `03_seed_data.sql` - Initial data population

## Usage

Run these scripts in order in your Supabase SQL editor:

1. Run `01_tables.sql` to create all tables
2. Run `02_functions.sql` to set up functions and triggers
3. Run `03_seed_data.sql` to populate initial data (optional)
